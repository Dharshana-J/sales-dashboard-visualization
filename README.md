# Sales Dashboard Visualization

## Project Overview

This project focuses on visualizing sales data using Python and Matplotlib. The objective is to transform raw business data into meaningful visual insights through different types of charts and graphs.

## Dataset Information

The dataset contains monthly business performance data, including:

- Month
- Sales
- Profit
- Orders

## Tools and Technologies

- Python
- Pandas
- Matplotlib

## Visualizations Created

### 1. Sales Trend Analysis
A line chart was used to visualize monthly sales performance and identify growth trends over time.

### 2. Profit Contribution Analysis
A pie chart was created to show the percentage contribution of profit across different months.

### 3. Orders Analysis
A bar chart was used to compare the number of orders received each month.

### 4. Sales vs Profit Relationship
A scatter plot was created to analyze the relationship between sales and profit.

## Key Insights

- Sales showed a steady increase throughout the year.
- Higher sales months generally generated higher profits.
- Order volume increased alongside sales growth.
- Business performance was strongest during the final quarter of the year.
